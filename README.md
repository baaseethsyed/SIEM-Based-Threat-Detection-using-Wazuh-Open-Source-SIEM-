# SIEM Project using Wazuh (Open Source)

## Overview
This project demonstrates a SIEM implementation using Wazuh for log monitoring and threat detection.

## Features
- Brute-force attack detection
- Failed login monitoring
- Real-time alerting

## Tools Used
- Wazuh
- ELK Stack
- Sysmon

## How to Run
1. Install Wazuh Manager
2. Configure Agent using ossec.conf
3. Add custom rules
4. Run attack simulation script
